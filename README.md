# Ejercicio: Dibujar la letra U

Completa el código en `DibujarLetraU.java` para que se dibuje la letra U en una matriz de 7x5 usando arrays bidimensionales.

## Objetivos
- Usar arrays bidimensionales en Java.
- Aplicar bucles `for` anidados.
- Comprender cómo representar gráficos simples en consola.

## Requisitos
- La letra U debe tener dos barras verticales y una base horizontal.
- Usa el carácter `'*'` para dibujar y `' '` para los espacios.

## Entrega
- Sube tu solución al repositorio antes de la fecha límite.
